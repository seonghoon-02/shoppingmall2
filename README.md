| 홈 화면 | [1] 상품 목록 보기 | [2] 장바구니에 담기 | [3] 장바구니에 담긴 상품의 총 가격 보기 | [4] 프로그램 종료 | [6] 장바구니 비우기<숨겨진 기능> |
